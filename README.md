# Java/FXML CSV File Reader GUI

- User inputs '.csv' file which contains the Payroll data for 2022 Massachusetts State employees.
- The data is displayed as GUI & could be sorted by 'job title, transaction id, name, payroll average for job, 'top5 & bottom 5 paid employees for job, etc'.
- Before running, un-zip '2022-payroll.csv.zip' in 'gradle/resources/main' and resources.
